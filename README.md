# PocketBook - Expense Tracker Dashboard

PocketBook is a **simple and interactive web-based expense tracker** that helps users monitor their income, expenses, and daily spending trends. The dashboard provides a clean UI with dynamic graphs and dark/light mode support.

## Features

- **Add Transactions:** Add income or expense with a description and amount.
- **Dashboard Summary:** See total balance, income, and expense in real-time.
- **Transaction History:** Keep track of all recent transactions.
- **Dynamic Graphs:** Visualize spending trends and category-wise expenses.
- **Dark/Light Theme:** Toggle between dark and light modes.
- **Responsive Design:** Works well on both desktop and mobile screens.
- **Edit/Delete Transactions:** Easily update or remove any transaction.
